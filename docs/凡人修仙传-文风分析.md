

## StyleOS

### 1. 作品定位

| 维度 | 配置 |
|------|------|
| **题材** | 古典仙侠 / 凡人流修真 / 升级闯关 |
| **世界观** | 层级分明的修仙界（凡人→练气→筑基→结丹→元婴→化神→炼虚→合体→大乘→真仙），宗门、散修、魔道、妖族、灵界、仙界多地图递进 |
| **目标读者** | 18-35岁男性网文读者；偏好理性主角、慢热升级、智斗生存、资源博弈 |
| **核心卖点** | 平庸资质+谨慎性格+奇遇道具（某秘宝）的反差；以弱胜强、扮猪吃虎、夺宝逃生 |
| **情绪价值** | 安全感（主角总能逃脱）、掌控感（信息差碾压）、成就感（境界突破）、紧张感（生死一线） |

### 2. 叙事系统

**叙事视角：** 第三人称有限视角为主，紧贴主角；偶切对手/旁观者反应镜头。

**视角切换规则：**
- 默认绑定主角感官与认知，不写主角不可能知道的信息
- 每300-800字可切一次"对方却不知道……""二人心中均是一凛"式旁观/对手反应段
- 大战场景在主角与敌修/围观修士间交替，但单次切换不超过2段
- 全知补叙仅用于世界观/宗门历史，以"有句话说得好""相传"等引子嵌入，不超过一段

**时间推进方式：**
- 日常：线性顺叙，"第二天""一连过了两天""数月后"等时间词直跳
- 修炼：大段时间压缩为一两句（"苦修数载""在此洞中闭关了十余年"）
- 战斗：时间放慢，动作链细分到"一瞬""下一刻""随即"

**信息披露策略：**
- 设定从不独立科普段出现，一律嵌入对话、战斗、拾取宝物、阅读典籍中
- 先给现象后给解释：先写效果（"灵光一闪""面色大变"），再通过人物对话或内心补原理
- 保留信息差：常用"主角表面不动声色，心中却……"制造读者与配角的认知差

**悬念设计方式：**
- 章末钩子：新敌人现身、秘宝异动、生死未卜（"并不知道……已经来了一位改变命运的客人"）
- 过程悬念：丹药是否成、法阵能否破、对方底牌几何
- 长线悬念：某秘宝来历、某宿敌阴谋、飞升之路

**伏笔布置方式：**
- 道具伏笔：早期提及的符箓、丹药、傀儡后在关键时刻兑现
- 人物伏笔：看似路人修士后章成为强敌或盟友
- 台词伏笔：前辈随口一言、古籍半句记载，数十章后呼应

**剧情推进模式：**
```
遇险/受辱 → 隐忍观察 → 搜集情报与资源 → 布局或逃亡 → 反杀/脱身 → 收获整理 → 进入更大地图
```
循环嵌套，地图逐级放大。

**常见章节结构：**
1. 承接上章悬念（1-3段）
2. 新场景/新人物出场（3-8段）
3. 冲突酝酿：对话试探或环境异变（10-20段）
4. 冲突爆发：打斗或智斗（15-40段，中后期章占比更高）
5. 收尾：战利品清点+伤势处理+下一步打算（3-8段）
6. 可选章末钩子（1-2段）

**典型节奏：** 慢热铺垫（40%）→ 骤然紧张（20%）→ 高潮爆发（25%）→ 冷静善后（15%）

**每1000字平均发生事件数：** 约2-3个叙事节拍（以"顿时/随即/忽然/蓦然/竟然/下一刻"等转折词及场景切换计，抽样统计约1.9处/千字，叠加对话交锋和动作回合约2.5个/千字）

### 3. 语言系统

**整体语言风格：** 白描为骨、轻修辞；平实准确的动词名词推动画面；书面语为主，带少量口语（早期凡人篇）。

**语言颗粒度：** 中颗粒（比纯动作流细，比心理流粗）

**平均句长：** 32字/句（全样本）；早期凡人篇约22-28字，修仙战斗段约25-35字，后期约30-38字

**短句占比：** 约20%（≤15字）；战斗高潮段升至35-45%

**长句占比：** 约29%（≥40字）；多用于设定交代、环境铺陈

**段落长度特征：**
- 平均段落53字；早期章节约67字
- 66.7%段落仅含1句；26.9%含2-3句；6.4%含4句以上
- 短段（≤20字）约10%；中段（20-80字）约75%；长段（80字+）约15%

**断句习惯：**
- 全角空格（　　）段首缩进
- 逗号密集切分动作链："单手一翻转，一个洁白无暇的小瓶出现在了中，临大敌的对准飞来的刀光就瓶口倒"
- 战斗段多用顿号并列动作

**口语化程度：** 低-中（早期某长辈、村民对话偏口语；修仙界对话偏书面）

**书面化程度：** 高（"道友""前辈""晚辈""不假思索""不动声色"）

**常见语气：** 冷淡、克制、略带讥讽（反派/主角后期）

**常见连接词：** 就（最高频）、而、也、但、还、却、才、又；双字：只是、不过、竟然、接着、顿时、但是、忽然、看来、随后、果然

**常见转场词：** 就在这时、与此同时、另一边、远处、此刻、这时、接着、随后、片刻后、下一刻、顿时

**常见强调方式：**
- 副词叠用："微微""淡淡""缓缓""冷冷"
- 程度词："竟""竟然""居然""赫然"
- 短句独立成段强调结果

**常见重复句式：**
- "一声……，……随即……"
- "……一声……，……就……"
- "不禁……起来"
- "……之下，……"
- "仿佛……一般"

**高频表达模板：**
- `[人物]闻听此言，[神态]，[动作/回应]`
- `[人物]心中一凛/一动，[应对]`
- `[人物]面无表情/淡淡的说道`
- `话音刚落，[异变]`
- `见此情形，[人物][反应]`

**高频句式模板：**
- `……，却见……`
- `……，竟……`
- `……，随即……`
- `……，不由得……`
- `……，心中暗暗……`

### 4. 场景描写系统

**场景描写频率：** 每章2-5处环境定调；战斗场景中低频率点睛（约每500字0-1处）

**场景描写长度：** 1-3句为主，极少超过80字

**描写重点：** 光线、声音、气味、空间尺度；修炼场景重灵气/法阵；凡人场景重贫穷细节

**感官比例（估算）：**
| 视觉 | 听觉 | 嗅觉 | 触觉 | 动态 | 静态 |
|------|------|------|------|------|------|
| 55% | 20% | 8% | 7% | 45% | 55% |

**环境与情绪联动规律：**
- 压抑→阴暗通道、霉味、寒气
- 危险→灵气波动、嗡鸣、灵压
- 安全→洞府、禁制、灵泉
- 凡俗→破旧、烟火气、旱烟味

**氛围塑造方法：** 一两处感官细节定调，不铺陈；用声音（打呼、嗡鸣、惨叫）制造临场感

**常见意象：** 灵光、遁光、血云、剑光、法阵符文、灵压、虚空裂缝、灵石、丹药香气

**高频环境词：** 灵光、光芒、虚空、法阵、洞府、山峰、灵气、灵压、血雾、青光、金光、银光

**场景描写模板：**
```
[地点]中，[静态景物]。[感官细节1]，[感官细节2]。
[人物]四下略一打量，[反应]。
```

### 5. 人物塑造系统

**人物出场方式：**
- 先声后形：对话/脚步声/灵压 → 外貌
- 群像出场：衣着分三类、站位分阵营
- 高手出场：灵压先行、虚空波动、遁光落地

**人物标签建立方式：**
- 一个外貌特征（"冷冷的面孔""圆脸小胡子"）
- 一个行为特征（"一言不发""眯起双眼"）
- 一个身份称谓（"道友""前辈""某真人"）

**外貌描写规律：** 1-2句；重神态不重美丑；修士重气质（"仙风道骨""凶神恶煞"）

**神态描写规律：** 面色/目中/嘴角三联；表情微变暗示心理（"似笑非笑""似怒非怒"）

**动作描写规律：**
- 战斗：法诀→灵光→法器→结果，链式推进
- 日常：简洁单一动作
- 高手：轻描淡写（"随手一抓""轻轻一点"）

**人物关系塑造方式：** 称谓体系（前辈/晚辈/道友）；利益交换对话；师徒/同门以功法丹药为纽带

**人物成长方式：** 境界突破外显+处事更谨慎冷酷；少写性格转变，多写手段增多

**人物反差设计方式：** 外表平凡/内心老辣；表面恭敬/暗中算计；强敌轻描淡写碾压

**高频动作：** 身形一闪、遁光、单手一掐诀、袖袍一抖、双目一眯、抬手、抓、喷出、化为、盘旋、没入

**高频神态：** 面无表情、冷笑、一怔、目中寒光、瞳孔一缩、面色微变、似笑非笑、阴沉

**高频身体反馈：** 心中一凛、不禁、倒吸一口凉气、面色苍白、铁青、额头见汗、身子微颤

**人物塑造模板：**
```
[称谓][外貌一两句]，此刻[神态]，[动作]。
[对话或内心]，显然[性格标签]。
```

### 6. 对话系统

**日常对话特征：** 简洁、带称呼、信息量大；少寒暄

**冲突对话特征：** 短句对峙、威胁与试探交替、话中有话

**对话长度：** 单句平均15-35字；一轮交锋2-6句

**废话比例：** ≤15%

**信息密度：** 高（境界、来历、底牌、条件常藏对话中）

**人物辨识度：** 靠称谓+语气（冷哼/轻笑/厉声）+立场，不靠方言

**网络语言：** 无

**方言：** 极少（早期村民口语）

**常见语气词：** 哼、咦、啊（惊讶）、罢了、而已

**高频口头禅：** "道友说笑了""不足挂齿""在下""前辈慧眼如炬""看来""莫非"

**高频对话模板：**
- `"[称呼]，[试探/威胁]？"`
- `"[人物]冷笑一声，"[反击]。"`
- `"哼！"[人物][动作]，"[不屑回应]。"`
- `"不错/的确如此，[补充信息]。"`

**冲突升级模式：** 言语试探 → 冷哼/威胁 → 亮法宝 → 动手

**打脸模式：** 对手嘲讽 → 主角隐藏实力 → 一招制敌 → 围观者震惊（"倒吸一口凉气"）

### 7. 心理描写系统

**心理描写频率：** 中低；约每300-500字一处

**平均长度：** 1-2句，极少超过50字

**理性分析占比：** 约70%（利弊权衡、脱困算计）

**情绪表达占比：** 约30%（恐惧、愤怒、庆幸）

**内心独白风格：** 冷静、算计型；早期少年篇略带感性

**心理活动触发机制：** 危机、利诱、识破骗局、见到强敌、收获异宝

**心理描写模板：**
- `心中一凛，[判断]`
- `心中一动，[计策]`
- `不禁[情绪动词]起来`
- `暗暗叫苦/暗暗叫了一声侥幸`
- `表面不动声色，心中却[真实想法]`

**高频心理词：** 心中、不禁、暗想、暗暗、心中一凛、心中一动、心中大定、叫苦、侥幸

### 8. 情绪系统

**主情绪：** 紧张（生存压力）

**辅助情绪：** 谨慎、庆幸、冷峻快意（反杀后）

**情绪推进规律：** 低起伏日常 → 陡升危机 → 高潮爆发 → 快速回落为冷静盘点

**情绪波动幅度：** 整体克制；大起大落落点少，以"心中一凛"代替嚎啕

**常见情绪曲线：**
```
平静（修炼/赶路）→ 警觉（遇敌）→ 恐惧/算计（陷入绝境）→ 爆发/逃脱 → 冷静（清点战利品）
```

**爽点设计方式：**
- 隐藏实力一招翻盘
- 夺宝得手全身而退
- 境界突破天地异象
- 强敌轻视→反被打脸

**虐点设计方式：**
- 被迫受制（某宿敌、禁制、威胁家人）
- 同伴陨落（轻描淡写一笔带过）
- 资源匮乏、资质平庸

**共鸣点设计方式：** 凡人出身、家族牵挂、步步为营

**读者情绪操控策略：** 信息差（读者知主角底牌而对手不知）+ 延迟满足（压抑后爆发）

### 9. 冲突系统

**主要冲突类型：** 修士争斗、夺宝、宗门倾轧、正魔对立、生存逃生

**次要冲突类型：** 阵法破解、炼丹炼器、境界突破、人情债务

**冲突来源：** 资源稀缺、怀璧其罪、立场对立、阴谋暗算

**冲突升级规律：** 口角 → 试探出手 → 法宝对轰 → 底牌尽出 → 分生死

**冲突解决规律：** 主角多以遁逃、符箓、傀儡、阵法脱身，少硬拼到底；必杀人时斩草除根

**代价设计：** 法力损耗、宝物损毁、受伤、暴露底牌、结仇

**风险设计：** 境界压制、人数围攻、禁制困杀、时间紧迫

**反转设计：** 看似必死→隐藏后手→局势逆转

**冲突模板：**
```
[敌方]冷笑一声，[出手]。
主角面色不变，[暗中准备]。
[交锋2-5回合]。
[结果：逃脱/反杀/两败]。
```

### 10. 回忆与闪回系统

**回忆触发方式：** 见到旧物、旧地、旧人；修炼功法联想

**回忆出现频率：** 低（大段回忆罕见）

**回忆长度：** 1-5段，用于补全背景

**回忆作用：** 解释功法来历、人物恩怨、早期伏笔回收

**过去线与当前线衔接：** "主角想起当年……""昔年在某宗门时……"

**闪回模板：**
```
[现实触发物/场景]。
主角目光微闪，想起了[时间]在[地点]发生的[事件]。
[回忆1-3段]。
回过神来，[回到当下行动]。
```

### 11. 修辞系统

**比喻使用频率：** 极低（约0.5次/千字；"仿佛"为主）

**拟人使用频率：** 低（法器有灵性例外）

**排比使用频率：** 极低

**夸张使用频率：** 低-中（用于形容灵压、神通规模）

**反讽使用频率：** 中（对话中冷嘲）

**象征使用频率：** 低

**常见意象：** 剑光、血雾、灵光、星辰、骄阳、飓风

**常见比喻源：** 自然物（风、云、骄阳、闪电）；极少跨域比喻

**修辞模板：**
- `仿佛[具体物]一般`
- `犹如[自然现象]`
- `宛如[实体]`

### 13. AI复现规则

**必须保留：**
- 白描动词驱动，形容词克制（每段≤2个修饰语）
- 第三人称贴主角视角+偶切对手反应
- 设定嵌入对话/行动，禁止独立科普段
- 冷静克制的叙述温度
- 战斗动作链：法诀→灵光→法器→结果
- 以弱胜强时的信息差与后手逻辑
- 章末悬念钩子
- 全角缩进段落格式

**可以弱化：**
- 后期重复的境界介绍
- 过长的拍卖会流水账
- 部分丹方/材料清单细节

**必须避免：**
- 华丽比喻堆砌、抒情散文腔
- 主角热血喊话、中二宣言
- 直接大段心理独白
- 现代网络梗、吐槽体
- 玛丽苏式外貌描写
- 冲突靠巧合解决（须有前置铺垫）
- 煽情离别、嚎啕大哭

**节奏规则：**
- 段内1句为主（≥65%）
- 每段必推进：情节/信息/情绪/悬念四选一
- 战斗：3-5个动作后接1句反应/结果
- 日常可放缓，但每500字须有1个小转折

**句式规则：**
- 核心句长15-40字；战斗压缩至8-20字
- 多用"……，随即/顿时/竟然……"
- 段首不用相同词连续超过2次

**对话规则：**
- 带称谓（道友/前辈/晚辈）
- 冲突对话≤3句一轮
- 对话后必接神态或动作

**描写规则：**
- 环境描写≤3句/处
- 感官细节点睛即止
- 大战场面轻描淡写（"轻轻一点""随手一抓"）

**情绪规则：**
- 用"心中一凛""面色微变"代替直接情绪词
- 生死关头不煽情
- 爽点靠反转和实力展示，不靠形容词

**冲突规则：**
- 先试探后全力
- 主角优先保命再杀敌
- 杀人须斩草除根，不留后患

**人物规则：**
- 新角色1-2个标签立住
- 反派先傲后惊
- 主角：表面恭顺/平静，内心算计

**场景规则：**
- 新地图首段给1-2句环境定调
- 洞府/秘境重禁制与灵气
- 凡人场景重烟火气

### 14. 风格DNA总结（≤2000字）

**最核心的文风特征：** 白描为骨。作者以平实动词和名词搭建修仙世界，极少修辞装饰。文字像摄像机的固定机位：客观记录动作、光影、声音，情感由读者自行推断。这是"冷"的文字。

**最核心的文笔特征：** 短句造势、长句铺陈。战斗时压缩为"一抬手""灵光一闪""顿时爆裂"的短链；设定与背景用30-50字长句嵌入叙述。副词"微微""淡淡""缓缓"是温度调节阀——越强大的存在，动作越轻描淡写。

**最核心的叙事习惯：** 贴主角视角的信息管控。读者与主角同步认知，偶尔切到对手视角制造"他不知道"的张力。剧情靠对话和行动推进，不靠旁白说教。每一段落都有存在理由——推进、揭示、转折、悬念，无废笔。

**最核心的情绪驱动方式：** 生存焦虑。不是热血，是"如何活着离开"。情绪表达高度间接：铁青、一凛、倒吸凉气、瞳孔一缩，替代直写恐惧愤怒。爽感来自算计得逞和实力碾压的反差，不是喊口号。

**最核心的读者吸引机制：** 凡人流共鸣（资质平庸、资源匮乏）+ 奇遇变强（某秘宝）+ 谨慎智商在线（不送死、不圣母）+ 地图无限升级（永远有更强大的世界）。

**AI模仿必须抓住的关键点：**
1. 白描动词链，形容词密度<8%
2. 主角式内心：利弊分析>情绪宣泄
3. 设定从对话和战斗中自然流出
4. 战斗描写：法诀→光效→结果，轻描淡写显高手
5. 章末必留悬念
6. 冷静叙述温度贯穿始终

**AI模仿最容易出错的地方：**
1. 写成热血少年漫式呐喊
2. 比喻排比过多，失去白描质感
3. 独立科普段破坏叙事流
4. 心理描写直白冗长
5. 战斗描写华丽但逻辑空洞
6. 对话现代化、吐槽化
7. 段落过长、句数过多，失去节奏
8. 煽情离别段落

## 代表性范文（脱敏保留原文风格）

### 范文一：开篇生活描写 —— 白描定调、以小见大

```
　　少年睁大着双眼，直直望着茅草和烂泥糊成的黑屋顶，身上盖着的旧棉被，已呈深黄色，看不出原来的本来面目，还若有若无的散发着淡淡的霉味。
　　在他身边紧挨着的另一人，是二哥，酣睡的十分香甜，从他身上不时传来轻重不一的阵阵打呼声。
　　离床大约半丈远的地方，是一堵黄泥糊成的土墙，因为时间过久，墙壁上裂开了几丝不起眼的细长口子，从这些裂纹中，隐隐约约的传来母亲唠唠叨叨的埋怨声，偶尔还掺杂着父亲抽旱烟杆的"啪嗒""啪嗒"吸吮声。
```

**要素：** 感官白描（霉味、打呼、旱烟）；贫穷细节；长句铺陈；无抒情。

### 范文二：心理克制 —— 行为代替情绪

```
　　在马车上，看着父母渐渐远去的身影，主角咬紧了嘴唇，强忍着不让自己眼眶中的泪珠流出来。
　　他虽然从小就比其他孩子成熟得多，但毕竟还是个十岁的小孩，第一次出远门让他的心里有点伤感和彷徨。
```

**要素：** 动作（咬唇）代替哭；心理简短；克制温度。

### 范文三：对峙对话 —— 信息+威胁

```
　　"不用再说什么，我也不想知道你以往的所作所为是真是假。好好听着，现在我只问你一句话：再给你一年时间，你能把功法练至第四层吗？"老者冷笑一声，徐徐地说出了今天最重要的一句话，接着眼皮眨都不眨，死死地盯住了少年，慎重的等着他的回答。
　　少年很清醒，知道这个问题的回答不但关系到对方一年后的性命，更是自己此时能否活下来的关键。
```

**要素：** 对话推进；神态（冷笑、盯）；心理一句点明利害。

### 范文四：战斗白描 —— 轻描淡写、动作链

```
　　青年目中寒光顿现，猛然一张口，两道刺目金芒射出，一晃后就诡异的到了血云面前，瞬间洞穿而过，不见了踪影。
　　两团血云中的惨叫声一起，两人直接从云中坠落而下。
　　就在此时，青年面无表情的冲他们轻轻一点。
　　两口金色小剑从附近虚空中闪出，只是围着他们轻轻一绕，一团团金芒绽放开来，瞬间将二人身形淹没进了其中，大片的血雨洒落而下。
```

**要素：** 短句连击；"轻轻一点"举重若轻；法器→结果；无华丽比喻。

### 范文五：后期智斗 —— 冷静试探

```
　　"证据？可笑，不是那人的话，你就去死吧！"主角脸上寒色一闪，一声厉喝后，一柄寸许大金色小剑从口中喷出，化为一道金光狠狠斩向某散修。
　　主角却眯着眼睛的看着空中发生的一切，神色动也不动，仿佛空中被攻击的并非是他本命法宝一般。
```

**要素：** 话少事大；神态冷静；反差（本命法宝被攻仍不动）。
