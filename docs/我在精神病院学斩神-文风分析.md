
### 2. 叙事系统

**叙事视角：** 第三人称有限视角；主绑甲，频繁切队友/反派/路人。

**视角切换规则：**
- 默认跟随甲的感官与判断
- 用空行+省略号段硬切，不写过渡（蒙太奇跳切）
- 战斗时可切旁观路人（"路人甲整个人都呆在了原地"）
- 双线并行：牺牲线/救援线交叉剪辑，用省略号段隔开

**时间推进方式：**
- 日常：顺叙，短章快切
- 战斗：慢镜头式细分动作帧
- 回溯：【某能力】式画面倒流，或回忆闪回

**信息披露策略：**
- 路人对话侧写引入主角（不直接介绍）
- 能力/设定通过战斗首次展示，不独立科普
- 悬念：身份、真相、内鬼逐层揭开

**悬念设计方式：**
- 开篇视觉冲击（黑缎缠目少年等反差画面）
- 章末异变/新敌人/新能力
- "他还不知道……"式信息差

**伏笔布置方式：**
- 十年前旧案、疗养院复查、纹章誓词
- 照片/监控/能力回溯埋线索

**剧情推进模式：**
```
日常吐槽/任务 → 神秘事件 → 调查推理 → 战斗爆发 → 能力觉醒/升级 → 善后+新钩子
```

**常见章节结构：**
1. 场景定调（1-5个短段，环境或音效）
2. 对话/吐槽（10-30短段）
3. 冲突或战斗（15-40短段，动作帧化）
4. 情绪升华（誓词/牺牲/反转，3-10短段）
5. 省略号切场或章末钩子

**典型节奏：** 幽默日常（30%）→ 紧张升级（25%）→ 高燃战斗（30%）→ 悲壮/留白（15%）

**每1000字平均发生事件数：** 约3-5个节拍（短段密度高，跳切频繁）

### 3. 语言系统

**整体语言风格：** 电影分镜式；动画化动作描写；口语吐槽与史诗誓词并存

**语言颗粒度：** 细颗粒（1段1信息点）

**平均句长：** 27.5字/句；高燃时压缩至3-15字

**短句占比：** 约37%（≤15字）

**长句占比：** 约25%（≥40字）；用于环境铺陈

**段落长度特征：**
- 平均段落33.9字
- **84.1%段落仅含1句**
- 短段（≤20字）约33%；中段约63%；长段约4%
- **空行率约50%**（段间留白极强）

**断句习惯：**
- 每段独立成块，段间空行
- 拟声词+破折号可独立成段（"咚——！！！"）
- 省略号段独立成行
- 排比短句可连打（"黑缎缠目，盲杖在肩，左手蔬菜，右手扛油……"）

**口语化程度：** 高（吐槽、粗口、网络梗适度）

**书面化程度：** 中（誓词、神战段落偏书面史诗）

**常见语气：** 吐槽、冷幽默、中二、正经切换

**常见连接词：** 只是、不过、竟然、顿时、与此同时、看来、片刻后、随着

**常见转场词：** 就在这时、与此同时、……（硬切）、另一边、随着

**常见强调方式：**
- 破折号延伸音效（"轰——！""滴滴滴——！"）
- 省略号留白
- 短句排比
- 拟声词独立段

**常见重复句式：**
- `[单信息点]。`（独立段）
- `"……"甲[动作]。`
- `……，眼眸/双眸微眯。`
- `……，身形一闪。`
- `……`（省略号段）

**高频表达模板：**
- `[拟声]——！`
- `[人物]愣在原地。`
- `[人物]淡淡开口："……"`
- `片刻后，[异变]。`
- `【某能力】。`

**高频句式模板：**
- `……，缓缓开口。`
- `……，微微一愣。`
- `……，眼眸中闪过一抹[情绪]。`
- `……，身形瞬间[动作]。`
- `……，仿佛[比喻]。`

### 13. AI复现规则

**必须保留：**
- 1句1段，段间空行
- 电影分镜跳切
- 拟声词+破折号
- 省略号留白/切场
- 幽默与燃点无缓冲切换
- 动画化动作描写
- 对话个性化
- 誓词仪式高潮

**可以弱化：**
- 后期重复境界介绍
- 过长设定堆砌

**必须避免：**
- 长段叙述不切块
- 纯抒情无画面
- 战斗无动作帧
- 全员同一语气
- 幽默缓冲悲剧（应硬切）
- 古风仙侠腔（本作都市异能）

**节奏规则：**
- 84%单句成段
- 每段仅1信息点
- 每200-400字拟声/音效
- 每150-300字省略号
- 章均约2450字

**句式规则：**
- 核心8-25字
- 高燃3-8字
- 拟声独立段

**对话规则：**
- 短句为主
- 对话独立段或与动作同段
- 队友互怼

**描写规则：**
- 视觉动画化
- 环境≤3句开场
- 眼眸/身形高频

**情绪规则：**
- 冷热硬切
- 牺牲用内心独白+誓词
- 不煽情冗长

**冲突规则：**
- 战斗短段连打
- 能力名【】标注后释放

**人物规则：**
- 一句定性格
- 甲：冷静吐槽
- 教官：粗口热血

**场景规则：**
- 拟声开场
- 省略号切场

### 14. 风格DNA总结（≤2000字，脱敏）

**最核心的文风特征：** 电影分镜式网文。文字不是"讲述"，而是"剪辑"——每段一个镜头，段间空行是呼吸，省略号是转场，破折号是音效。读者阅读体验接近看动画/电影，而非传统小说。

**最核心的文笔特征：** 动画化动作+口语吐槽+史诗誓词三轨并行。日常段像喜剧脚本（路人吐槽、队友互怼）；战斗段像分镜脚本（红芒、斩击、身形一闪）；高潮段像诗歌（誓词拆碎、排比、短句砸地）。

**最核心的叙事习惯：** 侧写引入、硬切跳场。主角常由路人对话带出；场景切换不写"然后他们来到了"，直接空行+新场景首句。双线叙事用省略号交叉，牺牲与救援并行。

**最核心的情绪驱动方式：** 反差落差。上一秒搞笑，下一秒血腥；上一秒日常，下一秒誓词。读者情绪被反复拉扯，燃点因此更燃，虐点因此更痛。

**最核心的读者吸引机制：** 中二燃爽+吐槽解压+群像羁绊+能力体系可视化+神战史诗感。

**AI模仿必须抓住的关键点：**
1. 1句1段+空行
2. 拟声破折号
3. 省略号切场
4. 幽默燃点硬切
5. 动作帧描写
6. 对话即人设
7. 誓词仪式段落

**AI模仿最容易出错的地方：**
1. 段落合并变长
2. 缺少空行留白
3. 战斗写成叙述而非画面
4. 幽默与悲剧之间加过渡
5. 全员书面腔
6. 忘记拟声词
7. 誓词一次说完，不拆碎分布
8. 使用原著名词未脱敏

## 代表性范文（人名/专名已脱敏）

### 范文一：旁观者视角引入 —— 分镜开场

```
炎炎八月。

滴滴滴——！

刺耳的蝉鸣混杂着此起彼伏的鸣笛声，回荡在人流湍急的街道上，灼热的阳光炙烤着灰褐色的沥青路面，热量涌动，整个街道仿佛都扭曲了起来。

路边为数不多的几团树荫下，几个小年轻正簇在一起，叼着烟等待着红绿灯。

突然，一个正在吞云吐雾的小年轻似乎是发现了什么，轻咦了一声，目光落在了街角某处。

"阿诺，你在看什么？"他身旁的同伴问道。

那个名为阿诺的年轻人呆呆的望着街角，半晌才开口，"你说……盲人怎么过马路？"
```

**要素：** 拟声开场；1句1段；路人对话侧写；不直接介绍主角。

```
只见在马路的对角，一个穿着黑色短袖的少年正站在那，双眼上缠着厚厚几圈黑色布缎，严严实实的挡住了所有光线。

他的左手拎着满载蔬菜的廉价购物袋，右手拽着扛在肩上的导盲杖，像是在扛一根扁担，而在导盲杖的末端，一大桶黄澄澄的花生油正在阳光下闪闪发光！
```

## 【文风文笔 - 必需遵守】（速查卡，脱敏）

1. **电影分镜**：每段一个画面/动作/台词，段间空行
2. **1句1段**：≥80%单句成段，极少超3句
3. **对话即性格**：吐槽/冷幽默/中二/正经，每人语气分明
4. **幽默燃点硬切**：搞笑与悲壮/血腥之间不设缓冲
5. **拟声破折号**：`轰——！` `咚——！` `滴滴滴——！` 独立或段首
6. **省略号切场**：`……` 单独成段，蒙太奇跳切
7. **动画化动作**：红芒爆开、枪尖燃焰、身形一闪、帧感描写
8. **誓词仪式**：关键时刻短句排比，可拆碎分布于多段
9. **侧写引入**：路人对话带出主角，不直接介绍
10. **专名脱敏**：用甲/乙/某组织/【某能力】代替原名

## 【量化自检 - 必需遵守】

1. 单句成段率 ≥80%
2. 单句字数：日常8-25字，高燃3-8字
3. 段间空行：每段后空行
4. 拟声/破折号：每200-400字至少1次
5. 省略号：每150-300字至少1次
6. 含对话段落：≥30%（或阅读感受≥50%）
7. 每段仅1信息点
8. 场景跳切：每500-1500字至少1次（省略号/空行）
9. 幽默段与燃/虐段：硬切，无过渡段
10. 禁止：长段合并、原名泄露、古风腔、无画面战斗
