### 2. 叙事系统

**叙事视角：** 第三人称有限视角；主视角绑定女主，中后期频繁切换某豪门群像（男主甲/男主乙/某长辈/某少爷等）。

**视角切换规则：**
- 默认跟随女主的认知与情绪
- 用 `*` 单独成段作场景/视角分隔（每章1-4次）
- 切换至其他角色时不预告，直接段首点名（"某长辈挂了电话""男主乙起身走向阳台"）
- 允许短暂全知补叙（豪门背景、三年前觉醒经过），以独立回顾段出现

**时间推进方式：**
- 当下线性推进为主
- 回忆以"三年前""老爷子去世后"等时间锚点闪回
- 社交媒体/热搜作为时间节奏器（推送打断宁静）
- 章节短，单章常聚焦一个场景或一场对峙

**信息披露策略：**
- 双层叙事：表层豪门恩怨 + 里层书灵/标签/剧情规则
- 信息通过热搜、群聊、书灵播报、角色脑补四条线释放
- 悬念保留：某祖父之死真相、内鬼身份、男主是否觉醒

**悬念设计方式：**
- 章末热搜/电话/书灵提示
- "她并不知道……"式信息差（配角脑补豪门攻心计）
- 声明/阴谋突现（如某豪门断绝关系声明）

**伏笔布置方式：**
- 人物标签系统（"人物标签：貌美，恶毒，愚蠢"）
- 婚鞋、女主别苑、牡丹花等意象伏笔
- 男主乙暗中布局长线（声明、木马、内鬼）

**剧情推进模式：**
```
标签逼迫走剧情 → 女主摆烂/反操作 → 舆论热搜发酵 → 豪门多方博弈 → 意外破局 → 新标签/新修罗场
```

**常见章节结构：**
1. 场景定调（1-3段，环境或情绪）
2. 对话交锋/吐槽（10-25段，占比最高）
3. 书灵或热搜插入（1-3段）
4. 冲突升级或反转（5-15段）
5. 章末留白（沉默/省略号/热搜/`*`切换）

**典型节奏：** 轻快吐槽（35%）→ 修罗场拉扯（30%）→ 阴谋暗涌（20%）→ 情绪爆点（15%）

**每1000字平均发生事件数：** 约3-4个节拍（对话回合、热搜推送、视角切换、情节反转合计，高于传统仙侠慢节奏）

### 3. 语言系统

**整体语言风格：** 轻快口语化书面语；吐槽感强；带现代网感但不泛滥；女主内心毒舌外冷

**语言颗粒度：** 中细颗粒（比凡人修仙传更碎，比短视频体更完整）

**平均句长：** 24字/句（全样本）；对话句约12-25字

**短句占比：** 约35%（≤15字）

**长句占比：** 约15%（≥40字）；用于豪门背景补叙

**段落长度特征：**
- 平均段落32.8字（明显短于长篇仙侠）
- 75.5%段落仅含1句
- 短段（≤20字）约32%；中段（20-80字）约65%；长段（80字+）约3%

**断句习惯：**
- 全角空格（　　）段首缩进
- 对话独立成段或与说话人同段（`女主，"……"`）
- 省略号段单独成行表沉默/无语
- `*` 单独成段作硬切

**口语化程度：** 高（吐槽、俗语、语气词密集）

**书面化程度：** 中（豪门称谓、新闻标题体、心理诊疗用语并存）

**常见语气：** 慵懒、毒舌、不耐烦、冷笑、故作轻松

**常见连接词：** 忽然、不过、竟然、只是、看来、显然、顿时、随即、所以、于是

**常见转场词：** 这时、就在这时、话音刚落、与此同时、*`、另一边、正说着、没等……

**常见强调方式：**
- 连续问号/感叹号（"什么？！""……"）
- 破折号拟声（"叮咚——""嘟——嘟——"）
- 加粗式热搜标题（`#【爆】……#`）
- 重复词（"没病好，没病就好"）

**常见重复句式：**
- `"……"某某无言以对。`
- `女主："……" / 某某愣了愣，"……"`
- `……，神情/语气[形容词]。`
- `……，显然没料到……`
- `【昵称：群聊内容】`

**高频表达模板：**

- `[人物]冷笑/挑眉/皱眉，[对话]`
- `女主[慵懒/恹恹/不悦]，[毒舌台词]`
- `书灵："……[剧情警告]"`

### 14. 风格DNA总结（≤2000字）

**最核心的文风特征：** 元叙事豪门吐槽文。表层是A国顶级豪门的甜虐修罗场，里层是炮灰觉醒对抗剧情规则的喜剧反叛。文字轻快、段落短碎、对话密、吐槽密，读者始终有"吃瓜+怼标签"的双重快感。

**最核心的文笔特征：** 对话定人设，吐槽代抒情。作者极少用大段心理或环境堆砌，而是让角色在来回交锋中暴露性格和立场。女主的慵懒毒舌是全书语言温度计——越云淡风轻，越反衬曾经的伤。比传统言情更口语，比短视频体更完整。

**最核心的叙事习惯：** 双轨信息释放。一条线是豪门现实（热搜、声明、家宴、官司），一条线是书灵元规则（标签、抹杀、剧情）。两条线互相催化：现实事件触发标签，标签逼迫现实失控。视角以女主为主，但`*`硬切至某豪门群像，制造全知吃瓜感。

**最核心的情绪驱动方式：** 反叛式解压。读者爽点不在"被宠爱"，而在"拒绝被定义"——拒绝恶毒标签、拒绝痴恋人设、拒绝为男女主让路。摆烂不是消极，是最高级的反抗。虐点集中在被误解、被冷落、爷爷线；甜点多在修罗场和追妻火葬场。

**最核心的读者吸引机制：** 反套路觉醒+豪门吃瓜+多男主修罗场+嘴炮女主+炮灰联盟。标签系统提供游戏感，热搜提供现实感，群像提供连续剧感。

**AI模仿必须抓住的关键点：**
1. 女主毒舌摆烂，绝不傻白甜
2. 书灵/标签元层不可缺
3. 短段+高密度对话
4. 热搜/推送/群聊现代节奏器
5. `*` 视角硬切
6. 旁人脑补制造误会喜剧
7. 冷男主要短句，不是霸总吼

**AI模仿最容易出错的地方：**
1. 写成普通甜宠，丢失元叙事
2. 女主变娇弱依赖
3. 段落过长、叙述过多
4. 男主写成油腻霸总
5. 缺少吐槽和反讽
6. 豪门权谋过重变严肃正剧
7. 忘记热搜/手机等现代元素
8. 书灵变成普通系统音，无"恶毒标签"核心矛盾

## 【文风文笔 - 必需遵守】（速查卡）

1. **对话吐槽为骨**：高密度对话+女主毒舌摆烂，用嘴炮推进情节
2. **元叙事不可缺**：书灵/标签/剧情规则是本书灵魂，不可写成普通豪门文
3. **短段快节奏**：均段≤35字，75%单句成段，章均2300字左右
4. **现代节奏器**：热搜、推送、群聊、电话打断叙事
5. **`*`硬切视角**：场景/人物切换用`*`单独成段
6. **旁人脑补误会**：配角按标签误解女主，制造喜剧反差
7. **慵懒外冷内热**：女主表面无所谓，涉及爷爷/尊严时语气骤降冰点
8. **男主冷短句**：男主甲/男主乙克制，不是油腻霸总
9. **省略号表无语**："……"独立段或段尾，喜剧/沉默兼用
10. **修罗场并行**：多男主拉扯，尊重女主选择权

## 【量化自检 - 必需遵守】

1. 单段句数：1句/段≥75%，极少超3句
2. 单句字数：对话10-25字，叙述20-35字
3. 含对话段落：≥40%
4. 每300-500字：至少1次对话冲突/热搜/视角切
5. 元叙事词（书灵/标签/剧情）：每章至少出现2-3次（前期）
6. 吐槽语气：女主每章至少3处毒舌/摆烂表达
7. 省略号/`*`：每章至少各用1次
8. 神态链：对话后必接神情/动作
9. 禁止：古风腔、傻白甜、长篇环境、霸总吼
10. 信息推进：每段推进对话/舆论/关系/阴谋至少一项
