# 朱雀AI检测助手 — 检测规则逆向测试方案

## 目标

通过系统性的对比实验，全面摸清朱雀AI检测助手的检测维度、权重和阈值，为改写策略优化提供精确的规则模型。

## 已有实验总结

| 实验 | 朱雀结果 | 关键发现 |
|------|---------|---------|
| A1(凡人原文) | 100% 人工 | 人类基线 |
| F1(AI小说) | 0/85/15 (人/疑/AI) | AI长篇偏"疑似" |
| mix-ai/A1(口语AI+人类尾部) | 46/54/0 | 最佳混合结果 |
| mix-ai/B1(短对话AI) | 0/49/51 | 中等 |
| mix-ai/C1(精炼标准AI) | 0/28/72 | **最高AI分** |
| mix-ai/D1(超长碎片化AI) | 0/63/37 | 长度/碎片化降低AI分 |
| mix-ai/E1(短行断句AI) | 0/63/37 | 同D1类似效果 |

**已覆盖维度**：口语化、低频词替换、刻意错别字、句长均匀化、去标点、人类前置(15%/30%/35%)、交替混合、低频词+口语组合

**尚未覆盖的关键维度**：见下方测试矩阵

## 测试矩阵设计

### 第一大类：单一变量隔离实验（控制变量法）

以 A2(标准AI文本) 为基准，每次只改变一个维度，观察分数变化，精确定位每个因素的权重。

#### L系列：文本长度敏感性

| 编号 | 实验内容 | 目的 |
|------|---------|------|
| **L1** | A2原文截取前3段（~150字） | 短文本检测阈值 |
| **L2** | A2原文截取前8段（~500字） | 中短文本 |
| **L3** | A2原文完整（~900字） | 基准 |
| **L4** | A2内容扩写至2000字（同风格延续） | 长文本 |
| **L5** | A2内容扩写至4000字 | 超长文本 |
| **L6** | 纯AI生成5000+字完整章节 | 极长文本检测行为 |

**假设**：朱雀对短文本可能判断力下降；长文本可能被稀释为"疑似"

#### M系列：AI模型来源差异

使用相同prompt生成"清河镇进镇"场景：

| 编号 | 模型 | 目的 |
|------|------|------|
| **M1** | GPT-4o | OpenAI系列指纹 |
| **M2** | Claude 3.5 Sonnet | Anthropic系列指纹 |
| **M3** | Doubao-pro | 豆包系列指纹 |
| **M4** | DeepSeek-V3 | DeepSeek系列指纹 |
| **M5** | Qwen-Max | 通义千问系列指纹 |
| **M6** | GLM-4 | 智谱系列指纹 |
| **M7** | Llama 3.1 70B (本地) | 开源模型指纹 |

**假设**：朱雀可能对特定模型族的token分布更敏感

#### T系列：生成温度/采样参数

使用同一模型(GPT-4o)、同一prompt，只改temperature：

| 编号 | Temperature | 目的 |
|------|------------|------|
| **T1** | 0.0 | 最确定性输出 |
| **T2** | 0.3 | 低温 |
| **T3** | 0.7 | 默认温度 |
| **T4** | 1.0 | 高温 |
| **T5** | 1.3 | 极高温（更随机） |
| **T6** | 0.7 + top_p=0.5 | 受限采样 |
| **T7** | 0.7 + top_p=0.95 + freq_penalty=0.8 | 惩罚高频词 |

**假设**：高温输出更"不可预测"，可能降低AI分

#### P系列：段落结构隔离

使用A2的**完全相同文字内容**，只改变段落划分方式：

| 编号 | 段落策略 | 目的 |
|------|---------|------|
| **P1** | 所有文字合并为1个段落 | 无段落结构 |
| **P2** | 每句话独立成段 | 极碎片化 |
| **P3** | 按原文段落（基准） | 对照 |
| **P4** | 随机位置断段（3-7句一段，长短不一） | 模拟人类随意断段 |
| **P5** | 长短交替段落（1句/5句/2句/6句…） | 制造段落长度波动 |
| **P6** | 对话独立成段、叙述合并为大段 | 对话分离策略 |

**假设**：朱雀可能分析段落长度分布的均匀性

#### C系列：连接词/过渡词密度

基于A2，系统性增删连接词：

| 编号 | 策略 | 目的 |
|------|------|------|
| **C1** | 删除所有连接词（然而/因此/于是/不禁/仿佛等） | 零连接词 |
| **C2** | A2原文（基准） | 对照 |
| **C3** | 每隔2句插入一个连接词（密度约15%） | 中密度 |
| **C4** | 每句都有连接词（密度约50%） | 极高密度 |
| **C5** | 只保留口语化连接词（"结果/谁知/得了/这下"） | 口语连接 |
| **C6** | A1人类原文中的连接词密度作为目标 | 对标人类 |

**假设**：连接词密度可能是关键检测特征

#### S系列：句长分布

使用A2的语义内容，改变句子长度分布：

| 编号 | 句长策略 | 目的 |
|------|---------|------|
| **S1** | 所有句子控制在10-15字 | 极均匀 |
| **S2** | 所有句子控制在25-35字 | 均匀偏长 |
| **S3** | 标准差极大：交替出现3字句和40字句 | 人工极端波动 |
| **S4** | 模拟人类分布：正态分布μ=18, σ=12 | 模拟人类 |
| **S5** | 偏态分布：大量短句(60%<10字) + 少量长句 | 偏态 |
| **S6** | A1人类原文的实际句长分布作为目标 | 对标人类 |

**假设**：句长方差和分布形态是检测维度之一

#### V系列：词汇丰富度(TTR)

| 编号 | 策略 | 目的 |
|------|------|------|
| **V1** | A2原文（基准TTR） | 对照 |
| **V2** | 替换30%的词为同义罕见词 | 提高TTR |
| **V3** | 刻意重复用词（降低TTR到0.3以下） | 降低TTR |
| **V4** | 只替换动词为低频动词 | 动词多样性 |
| **V5** | 只替换形容词/副词为低频词 | 修饰语多样性 |
| **V6** | 插入方言词、品牌名、网络用语 | 词汇意外性 |

**假设**：TTR过低或缺少"意外词"可能触发AI判定

| **G2** | 都市言情 | 偶遇/重逢场景 |
| **G3** | 悬疑推理 | 案件现场描写 |
| **G4** | 科幻 | 未来世界描写 |
| **G5** | 恐怖灵异 | 诡异氛围营造 |
| **G6** | 历史架空 | 朝堂议事场景 |
| **G7** | 散文随笔 | 个人感悟/回忆 |
| **G8** | 新闻报道 | 虚构新闻稿 |
| **G9** | 学术论文摘要风格 | 模拟论文段落 |
| **G10** | 网络小白文 | 口水化爽文 |

**假设**：不同体裁的AI指纹密度不同，朱雀可能有体裁偏见

#### N系列：叙述视角

同一场景(林晚收猫爬架)，不同叙述视角：

| 编号 | 视角 | 目的 |
|------|------|------|
| **N1** | 第一人称（"我"） | 视角1 |
| **N2** | 第三人称有限（"林晚"） | 视角2(基准) |
| **N3** | 第三人称全知 | 视角3 |

| **N5** | 多视角切换（林晚→男生→林晚） | 复合视角 |

**假设**：第一人称写作可能更"人类化"，因为AI偏好第三人称全知

### 第三大类：AI指纹特征精确测试

把已知的AI指纹模式逐一注入人类文本(A1)，观察哪些模式最致命。

#### F系列：指纹注入（向A1人类文本注入AI特征）

| 编号 | 注入特征 | 注入量 | 目的 |
|------|---------|-------|------|
| **F1** | 模板情感句（"心中涌起一股"、"眼中闪过一丝"） | 每3段1句 | 测情感模板权重 |
| **F2** | 微动作描写（"眼睫一垂"、"指尖僵住"） | 每2段1处 | 测微动作权重 |
| **F3** | 比喻/仿佛句（"仿佛"、"宛如"、"像…似的"） | 每段1个 | 测比喻密度权重 |
| **F4** | 总结性收束句（"这一刻他明白了…"） | 每段末尾 | 测收束句权重 |
| **F5** | 三段式排比（"有的…有的…有的…"） | 2处 | 测排比权重 |
| **F6** | 电影镜头式动作链（"他转身→抬头→目光落在→嘴角微微上扬"） | 3处 | 测镜头链权重 |
| **F7** | 综合注入F1-F6所有特征 | 全量 | 测组合爆发效果 |
| **F8** | 将A1的句长方差压缩至AI水平(σ<5) | 全文 | 纯句长均匀化 |

**假设**：可以精确量化每个指纹特征对分数的贡献

#### R系列：指纹消除（从AI文本中移除特定特征）

基于F1(AI小说,猫爬架)，逐一移除AI特征：

| 编号 | 移除特征 | 目的 |
|------|---------|------|
| **R1** | 删除所有"仿佛/宛如/像…似的"比喻 | 测比喻对分数的拉高效果 |
| **R2** | 删除所有微动作描写 | 测微动作权重 |
| **R3** | 删除所有模板情感句 | 测情感模板权重 |
| **R4** | 替换所有高频连接词为口语词 | 测连接词替换效果 |
| **R5** | 打乱句长使之符合人类分布 | 测句长矫正效果 |
| **R6** | 同时执行R1-R5 | 全量消除 |
| **R7** | R6基础上 + 插入方言/口语词 | 全量消除+主动伪装 |

**假设**：可以找到"性价比最高"的消除策略

## 预期产出

1. **维度权重矩阵**：量化每个检测维度（词汇/句长/连接词/段落/模板/比喻等）对分数的边际贡献
2. **分类阈值曲线**：人类比例 vs 朱雀分数的映射关系
3. **位置敏感性图谱**：确认首尾/中间文本的权重差异
4. **最优改写策略**：基于实验结果，推导出最高ROI的改写规则组合
5. **模型差异报告**：不同模型的检测难度排名，指导模型选择
6. **检测规则模型**：拟合朱雀的评分函数，用于预测改写效果

## 统计分析方法

1. **单变量分析**：每个系列内部，绘制 变量值→AI分 曲线
2. **交互效应**：选取关键维度做2×2交叉实验（如：口语化 × 人类前置）
